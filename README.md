# 青原优品电商后台系统

本项目是一个纯 Java SE 实现的青海特色产品电商后台

系统使用 JDK 自带 `HttpServer`，不依赖 Spring、Maven 或第三方 jar。项目内置独立 JDK 到 `tools\jdk-21`，不会修改本机全局 Java 环境。业务数据保存在 `data\shop-data.bin`，商品图片保存在 `商品图片`，背景轮播图片保存在 `background`。

## 启动、停止与测试

启动服务：

```bat
run.bat
```

访问地址：

```text
http://localhost:8080/login
```

停止服务：

```bat
stop.bat
```

运行测试：

```bat
test.bat
```

如果双击 `run.bat` 后打不开页面，先确认没有旧服务占用端口，可以双击 `stop.bat` 后再运行 `run.bat`，或查看项目根目录下的 `server.log` 和 `server.err.log`。

## 页面结构

顶部状态栏包含平台名称、横向搜索栏和当前登录信息。普通用户导航栏居中显示：

1. 首页
2. 商品
3. 系统设置

管理员登录后，导航栏会在搜索栏下方显示完整后台入口：

1. 首页
2. 商品
3. 订单管理
4. 营销管理
5. AI分析
6. 账户管理
7. 系统设置

普通用户不会看到订单管理、营销管理、AI分析和账户管理入口，直接访问这些地址也会被权限拦截。

## 核心功能

- 商品展示：包含牦牛肉、冬虫夏草、三文鱼、黑枸杞、唐卡与刺绣、青稞酒、中藏药、藏毯、藜麦等青海特色商品。
- 图片轮播：每个商品绑定两张真实产品图片，在商品卡片和详情页循环播放。
- 商品管理：管理员可以新增、编辑和删除商品；新增或编辑时既可以填写图片地址，也可以直接选择本地图片文件上传，上传后的图片会保存到 `商品图片` 目录。
- 普通用户下单：普通用户可在商品卡片或详情页进入下单页面，填写用户姓名、联系方式、发货地址和数量，并选择微信支付或支付宝支付。
- 付款码展示：付款码图片位于 `pay_code` 文件夹，微信支付使用 `weixin.png`，支付宝支付使用 `zhifubao.jpg`。
- 订单管理：内置多条模拟订单，客户姓名、电话、地址和订单状态更接近真实后台数据。
- 订单同步：普通用户创建订单后状态为待处理；点击“我已完成付款”后状态更新为已付款，并同步显示到管理员的订单管理列表。管理员订单表会展示联系方式、收货地址和支付方式。
- 营销管理：饼图展示各品类销售额占比，并在图例中显示百分比；订单新增、取消或恢复后，刷新营销页即可实时反映变化。
- 账户管理：管理员可以查看账户列表，新增账户，编辑用户名、显示姓名、角色、启用状态，并可为账户重置密码。
- 系统设置：支持网页明暗模式、字体大小、界面风格切换，并支持当前用户修改用户名、显示姓名和密码。
- 背景轮播：页面使用 `background` 文件夹中的多张背景图循环切换，主要内容区域采用轻微透明效果。

## AI 分析

AI 功能使用 DeepSeek 的 OpenAI 兼容接口，管理员可在 AI 分析页面填写或更新：

```text
接口地址：https://api.deepseek.com/chat/completions
模型：deepseek-v4-flash
API Key：你的 DeepSeek API Key
```

AI 页面只保留交互界面，不显示“发送给 AI 的产品数据”。后台仍会自动把当前商品清单、分类、品牌、产地、价格、库存和订单数据发送给模型，用于分析产品结构、青海特色、库存风险和营销动作。
